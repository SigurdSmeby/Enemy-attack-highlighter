# Enemy Attack Highlighter

Highlights NPCs that are currently attacking your character.

## Features
- Automatically highlights NPC attackers (no manual tagging).
- Works even if you did not attack first.
- Configurable highlight mode: `outline`, `tile`, or `both`.
- Configurable highlight color and outline thickness.

## Development
- Java 11
- RuneLite dependency: `latest.release`
- Run client from IDE using the `run` gradle task.
